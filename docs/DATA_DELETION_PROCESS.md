# 資料刪除流程（Data Deletion Process）

最後更新：2026-03-08

本文件描述 Expense Tracker（iOS/Android）之資料刪除方式，供使用者與 App Store / Google Play 審核參考。

## A. 使用者可自行刪除（App 內）

使用者可在 App 中直接刪除：
- 單筆帳目
- 分類（封存）
- 訂閱資料
- 分期資料

> 刪除後即不再出現在本機資料中（除非使用者自行匯出備份）。

## B. 刪除整個本機資料

使用者可透過以下方式清除全部本機資料：
1. 解除安裝 App
2. （可選）重新安裝後不還原備份

## C. 帳號/雲端資料刪除申請（若同步功能啟用）

當雲端同步正式上線後，使用者可申請刪除伺服器端資料。

### 申請管道
- Email：`privacy@expense-tracker.app`（上架前請替換為正式信箱）
- 主旨：`[Data Deletion Request] Expense Tracker`

### 需提供資訊
- 註冊 Email（若有）
- 使用平台（iOS / Android）
- 刪除範圍（僅帳號 / 帳號 + 全部雲端資料）

### 處理時程
- 收到申請後 **7 個工作天內**完成處理並回覆。

### 刪除範圍
- 帳號識別資料
- 雲端同步中的帳務資料
- 與帳號綁定的設定資料

### 例外
基於法規或資安稽核需要，可能保留最小必要紀錄（如刪除請求處理紀錄），並於法規期限後自動清除。
